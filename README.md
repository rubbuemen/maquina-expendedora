# maquina-expendedora
Una simple máquina expendedora de refrescos

Módulos necesarios:
- npm i express
- npm i jsonwebtoken
- npm i nodemon
- npm i morgan
- npm i bcryptjs
- npm i dotenv
- npm i colors
- npm i cors
- npm i sweetalert2
- npm i bootstrap
